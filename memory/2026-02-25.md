# 2026-02-25 华为广告本体模型设计工作记录

## 工作内容概述

今天完成了华为广告本体模型设计的补充和优化工作，主要添加了广告行为实体、广告竞价实体以及汽车行业相关实体，使模型更加完整和实用。

## 主要改进

### 1. 广告行为实体添加

#### 1.1 广告请求实体
- 新增了广告请求实体，用于记录广告请求的详细信息
- 属性包括：请求ID、用户ID、设备ID、媒体版位ID、请求时间、网络环境等
- 建立了与广告竞价的关系

#### 1.2 广告曝光实体
- 新增了广告曝光实体，用于记录广告曝光的详细信息
- 属性包括：曝光ID、请求ID、广告任务ID、创意ID、曝光时间等
- 建立了与广告竞价和广告点击的关系

#### 1.3 广告点击实体
- 新增了广告点击实体，用于记录广告点击的详细信息
- 属性包括：点击ID、曝光ID、广告任务ID、点击时间等
- 建立了与广告曝光和广告转化的关系

#### 1.4 广告转化实体
- 新增了广告转化实体，用于记录广告转化的详细信息
- 属性包括：转化ID、点击ID、广告任务ID、转化目标ID、转化时间等
- 建立了与广告点击的关系

### 2. 广告竞价实体添加

#### 2.1 广告竞价实体
- 新增了广告竞价实体，用于记录广告竞价的详细信息
- 属性包括：竞价ID、请求ID、广告任务ID、竞价时间、出价金额、竞价结果、竞价排名等
- 建立了与广告请求和广告曝光的关系

#### 2.2 召回阶段实体
- 新增了召回阶段实体，用于记录广告竞价中的召回阶段信息
- 属性包括：召回ID、请求ID、召回任务列表、召回数量、召回时间等
- 建立了与广告竞价的关系

#### 2.3 粗排阶段实体
- 新增了粗排阶段实体，用于记录广告竞价中的粗排阶段信息
- 属性包括：粗排ID、请求ID、粗排任务列表、粗排数量、粗排时间等
- 建立了与广告竞价的关系

#### 2.4 精排阶段实体
- 新增了精排阶段实体，用于记录广告竞价中的精排阶段信息
- 属性包括：精排ID、请求ID、精排任务列表、精排数量、精排时间等
- 建立了与广告竞价的关系

### 3. 汽车行业相关实体添加

#### 3.1 汽车品牌实体
- 新增了汽车品牌实体，用于记录汽车品牌的详细信息
- 属性包括：品牌ID、品牌名称、品牌国籍、品牌定位等
- 建立了与汽车的关系

#### 3.2 汽车实体
- 新增了汽车实体，用于记录汽车的详细信息
- 属性包括：汽车ID、汽车名称、品牌ID、车型、价格区间、排量、驱动方式、燃油类型、上市时间、销量排名等
- 建立了与汽车品牌和汽车用户的关系

#### 3.3 汽车用户实体
- 新增了汽车用户实体，用于记录汽车用户的详细信息
- 属性包括：汽车用户ID、用户ID、汽车ID、购买时间、使用时长、保养记录等
- 建立了与用户和汽车的关系

### 4. 投放数据和指标实体添加

#### 4.1 投放数据实体
- 新增了投放数据实体，用于记录广告任务的投放数据
- 属性包括：数据ID、广告任务ID、日期、曝光量、点击量、转化量、消耗金额、CPC、CPM、CPA、CTR、CVR等
- 建立了与广告任务的关系

#### 4.2 指标实体
- 新增了指标实体，用于记录广告任务的计算指标
- 属性包括：指标ID、广告任务ID、日期、指标类型、指标值、计算方式、时间粒度等
- 建立了与投放数据的关系

### 5. 链接类型（关系）优化

#### 5.1 广告行为相关关系
- 新增了广告请求、广告竞价、广告曝光、广告点击、广告转化之间的关系
- 建立了广告请求 → 触发 → 广告竞价的关系
- 建立了广告竞价 → 包含 → 召回阶段、粗排阶段、精排阶段的关系
- 建立了广告竞价 → 结果 → 广告曝光的关系
- 建立了广告曝光 → 触发 → 广告点击的关系
- 建立了广告点击 → 触发 → 广告转化的关系

#### 5.2 汽车行业相关关系
- 新增了汽车、汽车品牌、汽车用户之间的关系
- 建立了汽车 → 属于 → 汽车品牌的关系
- 建立了用户 → 拥有 → 汽车用户的关系
- 建立了汽车 → 被拥有 → 汽车用户的关系

#### 5.3 投放数据相关关系
- 新增了广告任务、投放数据和指标之间的关系
- 建立了广告任务 → 产生 → 投放数据的关系
- 建立了投放数据 → 计算 → 指标的关系

## 总结

华为广告本体模型设计v2.0版本现在更加完整和实用，添加了广告行为实体、广告竞价实体以及汽车行业相关实体。这些补充使模型能够更好地支持广告投放策略的自动化生成和优化，特别是在汽车行业的投放策略分析方面。

广告行为实体的添加使我们能够更全面地分析广告投放的效果，包括请求、曝光、点击和转化等关键环节。广告竞价实体的添加使我们能够深入分析广告竞价过程，包括召回、粗排和精排等阶段。

汽车行业相关实体的添加使我们能够针对汽车行业的特点进行投放策略分析，包括汽车品牌、汽车车型、汽车用户等信息。这些信息将有助于广告主更精准地定位目标受众，提高广告投放的效果。

投放数据和指标实体的添加使我们能够对广告任务的投放效果进行量化分析，包括曝光量、点击量、转化量、消耗金额等关键指标，并通过计算得到CPC、CPM、CPA、CTR、CVR等重要指标，为广告投放策略的优化提供数据支持。

基于 Palantir Foundry 本体建模理念，华为广告本体模型设计v2.0版本通过明确的实体类型、链接类型、操作类型和接口，构建了一个完整的广告系统知识图谱。该模型支持广告投放策略的自动化生成和优化，能够更好地分析用户行为和内容信息，为广告主提供精准的投放策略。
